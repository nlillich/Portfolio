Portfolio

Simple Microsoft Windows Forms Alarm Clock

Alarm clock windows forms application made with C# that allows the user to add and edit alarms as well as change the alarm sound and snooze an alarm. Project follows the MVC design pattern with the use of delegates to separate the three components from one another.

Sudoku Solver Using Simulated Annealing

Sudoku solver that utilizes the functionality of simulated annealing to search for local solutions until a certain tempurature is reached of a final solution is found. The program runs fully in the console and is written using python. A reheating case is implemented for if the algorithm gets trapped at a local maximum before the final solution is found.
