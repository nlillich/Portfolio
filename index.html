Portfolio

Simple Microsoft Windows Forms Alarm Clock

Alarm clock windows forms application made with C# that allows the user to add and edit alarms as well as change the alarm sound and snooze an alarm. Project follows the MVC design pattern with the use of delegates to separate the three components from one another.

Sudoku Solver Using Simulated Annealing

Sudoku solver that utilizes the functionality of simulated annealing to search for local solutions until a certain temperature is reached or a final solution is found. The program runs fully in the console and is written using python. A reheating case is implemented for if the algorithm gets trapped at a local maximum before the final solution is found.

Pathfinder Maze Solver

Python program that utilizes five different search algorithms to solve various mazes. Program runs fully within the terminal and uses hardcoded text files to populate the mazes. The five search algorithms implemented are random search, depth-first search, breadth-first search, greedy search, and A* search.
